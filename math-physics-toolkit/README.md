# Math & Physics Helper Toolkit

A Python toolkit that helps students solve common Mathematics and Physics problems.  
This project includes modules for equations, trigonometry, projectile motion, Newton’s laws, forces, and more.

## Features
- Solve linear equations
- Solve quadratic equations
- Trigonometric calculations
- Projectile motion calculator
- Newton’s laws & force calculations
- Unit conversions
- User-friendly terminal interface

## How to Run
```bash
python main.py
